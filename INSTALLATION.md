# تعليمات التثبيت والاستخدام

## 🚀 البدء السريع

### الخطوة 1: فتح الموقع
افتح ملف `index.html` مباشرة في المتصفح بالنقر عليه نقراً مزدوجاً.

### الخطوة 2: تصفح الموقع
- اضغط على أقسام القائمة للتنقل بين الأقسام المختلفة
- شاهد أعمال المحترف في قسم الأعمال
- تصفح المتجر وأضف المنتجات للسلة
- استخدم نموذج التواصل للتواصل المباشر

## 📦 البنية الكاملة للملفات

```
workspace/
│
├── index.html                  ← الصفحة الرئيسية (ابدأ من هنا)
├── privacy-policy.html         ← سياسة الخصوصية
├── terms-of-service.html       ← شروط الاستخدام
├── meta-tags.html              ← وسوم SEO
├── robots.txt                  ← ملف للـ Search Engines
├── sitemap.xml                 ← خريطة الموقع
├── package.json                ← معلومات المشروع
├── README.md                   ← دليل المشروع
│
├── css/
│   └── style.css              ← جميع التنسيقات (CSS)
│
├── js/
│   ├── main.js                ← الوظائف الأساسية
│   └── advanced.js            ← الوظائف المتقدمة
│
├── images/                     ← مجلد الصور
│
└── assets/                     ← ملفات إضافية
```

## 🛠️ التعديلات المهمة

### 1. تعديل المعلومات الشخصية

في `index.html` ابحث عن:
- `محمود مصطفى` وغيرها بالاسم الخاص بك
- `+966 50 123 4567` بالهاتف الخاص بك
- `mahmoud@example.com` بالبريد الإلكتروني
- وسائل التواصل الاجتماعي

### 2. تعديل الأسعار

في قسم المتجر، غيّر الأسعار في:
```html
<span class="price">199 ريال</span>
```

### 3. إضافة صورك الخاصة

استبدل روابط الصور:
```html
<!-- من -->
<img src="https://via.placeholder.com/400x300?text=Project+1" alt="مشروع 1">

<!-- إلى -->
<img src="images/your-image.jpg" alt="مشروع 1">
```

### 4. تخصيص الألوان

في `css/style.css` عدّل:
```css
:root {
    --primary-color: #7c3aed;        /* اللون الأساسي */
    --secondary-color: #ec4899;      /* اللون الثانوي */
    --dark-bg: #0f172a;              /* الخلفية الداكنة */
}
```

## 🌐 النشر على الإنترنت

### خيار 1: استخدام Netlify (الأسهل)
1. قم بتسجيل حساب على https://netlify.com
2. اسحب ملفات المشروع إلى Netlify
3. سيتم نشر الموقع تلقائياً

### خيار 2: استخدام GitHub Pages
1. أنشئ repository على GitHub
2. أرفع الملفات
3. فعّل GitHub Pages من الإعدادات
4. الموقع سيكون متاحاً على `username.github.io`

### خيار 3: الاستضافة التقليدية
1. اشترك في خدمة استضافة
2. أرفع الملفات عبر FTP
3. أشر Domain على الاستضافة

## 🐛 حل المشاكل الشائعة

### الموقع لا يفتح
- تأكد من وجود الملفات في نفس المجلد
- تحقق من أسماء الملفات (حساسة لحالة الأحرف)
- جرب متصفح مختلف

### الصور لا تظهر
- استخدم الرابط الكامل للصور
- تأكد من امتداد الملف (.jpg, .png, .gif)
- تحقق من اسم الملف بدقة

### الأزرار لا تعمل
- افتح Console (F12) لمعرفة الأخطاء
- تحقق من أسماء الـ IDs
- تأكد من تحميل ملف JavaScript
