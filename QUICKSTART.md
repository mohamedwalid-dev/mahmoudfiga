# Guide de Démarrage Rapide / Quick Start Guide

## 🚀 بدء سريع - Start Quick

### العربية / Arabic

1. **افتح الملف**
   - انقر نقراً مزدوجاً على `index.html`

2. **تصفح الموقع**
   - استكشف الأقسام المختلفة من القائمة العلوية

3. **تخصيص الموقع**
   - عدّل الأسماء والأسعار والمعلومات الشخصية في `index.html`
   - غيّر الألوان في ملف `css/style.css`
   - أضف صورك الخاصة

4. **النشر**
   - احفظ النسخة الأخيرة
   - انشرها على Netlify أو أي خدمة استضافة أخرى

---

### English / الإنجليزية

1. **Open the File**
   - Double-click on `index.html`

2. **Browse the Website**
   - Explore different sections from the top menu

3. **Customize the Website**
   - Edit names, prices, and personal information in `index.html`
   - Change colors in the `css/style.css` file
   - Add your own images

4. **Deploy**
   - Save the final version
   - Deploy it on Netlify or any other hosting service

## 🎨 التخصيصات الأساسية

### 1. تغيير الاسم
ابحث عن `محمود مصطفى` واستبدله باسمك

### 2. تغيير الأسعار
ابحث عن `199 ريال` واستبدلها بأسعارك

### 3. تغيير الألوان
في `css/style.css` غيّر:
```css
--primary-color: #7c3aed;
--secondary-color: #ec4899;
```

### 4. إضافة صور
ضع صورك في مجلد `images/`
ثم غيّر رابط الصورة من:
```html
<img src="https://via.placeholder.com/..." />
```
إلى:
```html
<img src="images/your-image.jpg" />
```
