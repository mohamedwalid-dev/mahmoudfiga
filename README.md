# موقع محمود مصطفى - مصمم فيديو واحترافي المونتاج

موقع احترافي متكامل لعرض أعمال المونتاج والفيديو مع متجر إلكتروني لبيع المشاريع والقوالب.

## 📋 المميزات

### 🎬 عرض الأعمال
- معرض تفاعلي للأعمال السابقة
- تصنيفات متعددة (إعلانات، مقدمات، مونتاج عام)
- فلاتر سهلة الاستخدام
- صور عالية الجودة

### 🛒 المتجر الإلكتروني
- عرض متميز للمنتجات
- معلومات واضحة عن كل منتج
- أسعار محددة
- سهولة إضافة للسلة
- منتجات متنوعة:
  - قوالب فيديو احترافية
  - حزم انتقالات
  - مجموعات مؤثرات صوتية
  - دورات تعليمية

### 📊 إحصائيات
- عدد المشاريع المنجزة
- عدد العملاء الراضين
- سنوات الخبرة
- جودة الإنتاج

### 📱 تصميم متجاوب
- تصميم كامل للأجهزة المحمولة
- واجهة سهلة الاستخدام
- أداء سريع جداً

### 📞 التواصل
- نموذج تواصل متكامل
- معلومات التواصل الكاملة
- روابط الشبكات الاجتماعية
- خريطة الموقع

## 📁 البنية

```
workspace/
├── index.html           # الصفحة الرئيسية
├── css/
│   └── style.css       # ملف التنسيق الرئيسي
├── js/
│   └── main.js         # ملف JavaScript الرئيسي
├── images/             # مجلد الصور
└── assets/             # مجلد الملفات الإضافية
```

## 🚀 كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. تصفح الأقسام المختلفة
3. استكشف أعمال المحترف
4. تصفح المتجر وأضف منتجات للسلة
5. استخدم نموذج التواصل للتواصل المباشر

## 🎨 الألوان المستخدمة

- اللون الأساسي: `#7c3aed` (بنفسجي)
- اللون الثانوي: `#ec4899` (وردي)
- اللون الداكن: `#0f172a` (أسود مزرق)
- اللون الفاتح: `#f8fafc` (رمادي فاتح جداً)

## 📚 الأقسام الرئيسية

### 1. الرئيسية (Hero Section)
- عنوان جذاب
- وصف قصير
- أزرار دعوة للإجراء
- صورة أو فيديو بديل

### 2. أعمالي (Portfolio)
- معرض صور تفاعلي
- فلاتر حسب الفئة
- معلومات تفصيلية لكل مشروع

### 3. المتجر (Shop)
- عرض منتجات بتصميم جميل
- معلومات المنتج الكاملة
- أسعار واضحة
- زر "أضف للسلة"

### 4. معلومات (About)
- صورة شخصية
- نبذة تعريفية
- المهارات والخبرات

### 5. تواصل معي (Contact)
- نموذج تواصل
- بيانات التواصل
- روابط الشبكات الاجتماعية

## 🔧 تخصيص الموقع

### تعديل المعلومات الشخصية
غيّر البيانات التالية في `index.html`:
- الاسم والعنوان
- أرقام الهاتف والبريد
- روابط الشبكات الاجتماعية
- أسعار المنتجات

### تعديل الألوان
غيّر متغيرات CSS في `css/style.css`:
```css
:root {
    --primary-color: #7c3aed;
    --secondary-color: #ec4899;
    --dark-bg: #0f172a;
    --light-bg: #f8fafc;
}
```

### إضافة صور جديدة
1. ضع الصور في مجلد `images/`
2. غيّر رابط الصور في HTML من placeholder إلى صورك

## 🚀 نصائح للتحسين

1. **أضف صورك الخاصة** بدلاً من الصور الموجودة (Placeholder)
2. **أضف أعمالك الفعلية** للقسم الخاص بأعمالك
3. **حدّث أسعار المنتجات** حسب احتياجاتك
4. **ربط نموذج التواصل** بخدمة بريد إلكترونية حقيقية
5. **أضف تحليلات** مثل Google Analytics
